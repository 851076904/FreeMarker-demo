变量的赋值

<#assign a = 110 /> ${a}

算术运算

（1）加法： ${x} + ${y} = ${x + y}
（2）减法： ${x} - ${y} = ${x - y}
（3）乘法： ${x} * ${y} = ${x * y}
（4）除法： ${x} / ${y} = ${x / y}
（5）取模： ${x} % ${y} = ${x % y}
（6）获取除数的整数部分： ${x/y} => ${(x/y)?int}

比较操作（x=${x} y=${y}）

（1）= 或 ==，测试两个值是否相等： <#if x = y> <#-- x == y 等价 x = y --> x == y <#else> x != y
（2）!= 测试不相等： <#if x != y> x != y <#else> x == y
（3）>： <#if x gt y> (x > y) = YES <#else> (x > y) = NO
（4）>=： <#if x gte y> (x >= y) = YES <#else> (x >= y) = NO
（5）<： <#if x lt y> (x < y) = YES <#else> (x < y) = NO
（6）<=： <#if x lte y> (x <= y) = YES <#else> (x <= y) = NO

if 语法

<#assign b = 100 /> b = ${b}
<#if b??> <#if b > 50> b 大于 50 <#elseif b < 50> b 小于 50 <#else> b 等于 50

逻辑运算（ T = <#if T> true <#else> false F = <#if F> true <#else> false ）

（1）逻辑或 || ： ${T ? string("true", "false")} || ${F ? string("true", "false")} = ${ (T || F) ? string("yes", "no")}
（2）逻辑与 && ： ${T ? string("true", "false")} && ${F ? string("true", "false")} = ${ (T && F) ? string("yes", "no")}
（3）逻辑非 ! ： !${T ? string("true", "false")} = ${ (!T) ? string("yes", "no")}

switch

1、字符串 <#switch 'small'> <#case "small"> This will be processed if it is small <#break> <#case "medium"> This will be processed if it is medium <#break> <#case "large"> This will be processed if it is large <#break> <#default> This will be processed if it is neither
2、数字 <#switch 1> <#case 1> 1 <#case 2> 2 <#default> d
